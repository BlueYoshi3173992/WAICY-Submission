How To Use HandUI

HandUI is an experimental user interface that uses AI to determine hand gestures that dynamically interact with various UI elements. The experimental model was trained and created through Google's Teachable Machine platform.

HandUI uses your webcam to predict hand gestures in real-time, which are used as states to be mapped to different UI actions.

There are 6 unique gestures that the model recognizes:
  • Open Palm -> Await Instruction
  • Closed Fist -> Selected & Deselect
  • Pointing Up -> Move Element Selection
  • Pointing Down -> Move Element Selection
  • Pointing Right -> Increment Counter or Move Slider
  • Pointing Left -> Decrement Counter or Move Slider


Please make sure to deselect elements once you've selected them. Also, it is imperative that you check the hand signals with their corresponding weights to determine which signs are the clearest.

More information below:
Await (Wait) Instruction:
The await Instruction signals a break in instructions. This also means that the await instruction must be used between selection signals to prevent accidental multiple selections.

The gesture for this instruction is an open palm, held to the center of the webcam.
Select & Deselect Instruction:
The select and deselect instruction is used to interact with the selected UI element. For elements that take a binary input (like a button), the select instruction will invert the state. For every other element, the select instruction will toggle interaction with the element.

The gesture for this instruction is a closed fist, held to the center of the webcam.
Vertical Movement Instruction:
The vertical movement instruction is used to move the selection up or down. The demonstration UI will be a scrollable list, so for this demonstration only vertical movement will be allowed. It's only function is to move the selection vertically. This function will only enable after the await signal.

The gesture for this instruction is pointing either up or down (changes the movement direction), held to the center of the webcam.
Counter & Slider Instruction:
The counter and slider instruction is used to increment or decrement the value of a slider or a counter (both are functionally equal). This function will only enable after the await signal, given the slider or counter is selected.

The gesture for this instruction is pointing either right or left (changes whether to increment or decrement), held to the center of the webcam. Please be sure to hold fist tightly.
